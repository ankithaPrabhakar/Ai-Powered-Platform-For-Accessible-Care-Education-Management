üìå Abstract

Finding suitable education and care resources for children with disabilities is difficult due to the lack of centralized, personalized, and accessibility-focused platforms.

This project introduces an AI-powered web platform that provides:

location-based discovery of institutions,

feedback and rating systems,

and personalized recommendations using AI.

The platform bridges existing accessibility gaps and supports inclusive access to educational and care services, especially for underserved and marginalized communities.
